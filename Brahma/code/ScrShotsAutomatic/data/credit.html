Programmers
Brahma is the creation of the husband-wife programmer duo of Mayur & Shanmugapriya Kankanwadi. We have about 15 years of mobile app development experience between us. We have developed a lot of commercial mobile apps on our jobs, but this is our first hobby project to be open-source'd.
Client-side tools & technologies
Symbian IDE: Carbide.C++
HTML Editor : HTML-KIT
SVG Editor : Inkscape
CSS buttons created using Online CSS button generator.
Color combos for the app and html decided using Colors on the Web & ColorCombos.com.
The free WhirlPool Hash algorithm source code is also integrated, but will be only used in future releases.
Octin Stencil free font is used as a custom font on the screens.
The ttf to gdr conversion was done using Easy GDR Creator for S60.
